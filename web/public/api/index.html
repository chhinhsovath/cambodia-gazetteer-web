Endpoints បញ្ជី​ផ្លូវ

Path Returns
GET/api/v1/provinces List of all 25 provinces (codes, names).
GET/api/v1/provinces/{code} A province with its districts.
GET/api/v1/districts/{code} A district with its communes.
GET/api/v1/communes/{code} A commune with its villages.
GET/api/v1/search?q=…&type=…&limit=… Substring search across Khmer + Latin + code.
GET/api/v1/full.json The entire dataset as one nested JSON file.
GET/api/v1/openapi.json OpenAPI 3.1 spec.

Examples ឧទាហរណ៍

List provinces, then look up Banteay Meanchey:

$ curl https://cambodia-gazetteer-web.pages.dev/api/v1/provinces
$ curl https://cambodia-gazetteer-web.pages.dev/api/v1/provinces/01

Search for any place name containing “Bant”:

$ curl "https://cambodia-gazetteer-web.pages.dev/api/v1/search?q=bant&limit=5"

From the browser:

const res = await fetch("/api/v1/communes/10201");
const { data } = await res.json();
console.log(data.villages.length);  // 19

Download ទាញ​យក

Prefer the raw dataset? Download cambodia_gazetteer.json (~1.2 MB) — same data, original NCDD nested shape.

Caveats ចំណាំ

  • Read-only. Every response carries open CORS and is cached at the edge.
  • Hierarchical codes follow the NCDD scheme (province 01, district 000102, commune 10201, village 01020101).
  • The dataset is a snapshot of db.ncdd.gov.kh/gazetteer; village-level boundary changes since publication may not be reflected.
